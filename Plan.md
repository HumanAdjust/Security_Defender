# 임베디드
## 트레이싱기술(카메라)
    - 사람인식
    - 비밀번호

- 선에 감지 되었을때
  - 사진을 찍는다.
  - 웹에 보낸다.
  - c++ 에서 DB로 올린다.
  - 웹에서 소리가 난다.

# 웹 
    - 관리자 확인 (로그인)
    - 웹에서 받아와서 출력하기(날짜, 사진,+위치(방향))
    - 시간당 사진의 갯수
    - 경찰 호출
    - 선에 감지 되었을 때 서버로 넘어온 날짜와 시간을 보낸다.
- 데이터베이스에 값이 들어왔을때 발생하는 이벤트
- 값을 가져오면 소리가 난다.
